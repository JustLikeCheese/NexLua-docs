# NexLua

NexLua 使用 init.lua 作为项目的配置文件

```
-- 名称
appname="项目名称"
-- 包名
packagename="com.mycompany.testapp1"
-- 版本
appcode=1
-- 版本号
appver="1.0"
-- 最低 SDK 版本
appsdk="15"
-- 调试模式
debugmode=true
-- 应用权限
user_permissions={
    "android.permission.INTERNET",
    "android.permission.WRITE_EXTERNAL_STORAGE",
}

user_routes = {
    ["main.lua]
}
```
